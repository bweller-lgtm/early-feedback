# Example Reports

Real evaluation reports generated by the Early Feedback pipeline, included to demonstrate output quality and format.

| Report | Score | What it shows |
|---|---|---|
| [invoiceflow.md](invoiceflow.md) | 6.8/10 | A promising mid-score evaluation (freelancer invoicing tool). Balanced findings, actionable recommendations. |
| [brickdreams.md](brickdreams.md) | 4.4/10 | An honest low-score evaluation (AI brick set marketplace). Identifies fatal technical gap, suggests pivot. |

Both reports were generated with default settings: 8 personas, 3 experts, web research enabled.

## What to look for

- **Differentiated scores** -- 6.8 vs 4.4 shows the pipeline produces honest, varied assessments
- **Specific quotes** -- personas cite real tools, real prices, real workflows
- **Expert disagreements** -- experts independently reach different conclusions; tensions are preserved
- **Actionable recommendations** -- not generic advice, but specific next steps tied to findings
- **Confidence calibration** -- each report labels what's reliable vs. what needs real-world validation
